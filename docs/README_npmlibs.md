### npm on Macbook M1 pro (Serona)
1. Install nvm.

2. Source nvm
```
source ~/.nvm/nvm.sh
```

3. Install Node.js with nvm.
```
nvm install <version>
```

4. Use node version.
```
nvm use <version>
```

5. Install npm.
```
curl -L https://npmjs.org/install.sh | sh
```

Use version : 16

## Apollo setup
- You can install Apollo to generate queries.
```
npm install apollo-boost react-apollo graphql --save -f
```
