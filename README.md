# BookGraphQL
A sutdy demo uses GraphQL by programming node.js.

In the demo, we use Mongodb connecting to AWS.

For running Macbook M1 pro:

```
npx nodemon app
```

## Show

To check result, you should run on 4000 port.

```
localhost:4000/graphql
```
